# Pruebas-Gui-vs-Cli
Este repositorio fue creado con el objetivo de practicar y comprender el uso de Git y GitHub mediante dos enfoques:

- GUI (Graphical User Interface)
- CLI (Command Line Interface)

## Temas practicados

- Commit
- Branch
- Push
- Fetch
- Remote
- Origin
- Upstream
- Pull Request
- Fork
- Stash
- Submodule

## Objetivo

Comparar las ventajas y diferencias entre el uso de herramientas gráficas y la terminal para la gestión de repositorios Git.

## Herramientas utilizadas

- [Git](https://git-scm.com?utm_source=chatgpt.com)
- [GitHub](https://github.com?utm_source=chatgpt.com)
- [GitHub Desktop](https://desktop.github.com?utm_source=chatgpt.com)
